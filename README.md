# gulimall

#### 介绍
购物商城

#### 软件架构
本项目以分布式为基础，使用的springBoot快速开发，其中，mysql做持久化，redis做缓存，其中使用到微服务等一系列的组件，如nacos作为注册中心，以及配置中心，feign做远程调用。
